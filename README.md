## What is Snake Spectrum?

Snake Spectrum is an HTML arcade game that reimagines the classic snake formula as a smooth, modern experience in the browser. It is made for both desktop and mobile sessions, with responsive controls and a gameplay loop that is quick to learn while still supporting longer play sessions and repeated attempts.

The release offers 120 structured levels plus an endless mode for players who want ongoing challenge. With polished movement, collectibles, unlockable visual choices, and power-ups, it works well as a lightweight web game with plenty of variation.

## How to Play

Load the game in a modern browser and pick either level mode or endless mode. On mobile, use touch controls; on desktop, use arrow keys and any other supported keyboard input.

Typical flow:
1. Start a level or begin an endless run.
2. Collect items and power-ups as you move.
3. Avoid obstacles and manage the increasing pace.
4. Visit the shop to unlock visual options.
5. Replay stages or switch to endless mode for a different challenge.
